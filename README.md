E-Commerce Big Data Analytics

An end-to-end e-commerce analytics platform built with MongoDB, HBase, Apache Spark (PySpark), Docker, and Python to demonstrate distributed data storage and large-scale data processing.

Features
Generate a synthetic e-commerce dataset
Store data in MongoDB and HBase
Process data using Apache Spark
Revenue analysis
Top-selling products analysis
Customer segmentation
Product popularity insights
Tech Stack
Python
MongoDB
HBase
Apache Spark (PySpark)
Docker
